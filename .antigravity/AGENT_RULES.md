# Agent Rules for AI Assistants

This file contains mandatory system constraints for any AI agents interacting with this codebase.

## 1. Documentation Integrity (SDD)
If you ever modify the core logic, architecture, or lifecycle of the migrator script, you **MUST** update the following files in the same session to reflect the changes:
- `SPEC.md` (The Software Design Document)
- `README.md` (User guide and commands)

This ensures that the documentation is never out of sync with the implementation.
